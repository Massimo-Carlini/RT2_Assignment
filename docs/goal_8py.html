Detailed Description

package assignment_2_2022

This script creates a ROS node to handle the goal service and update the number of goals reached and cancelled. It creates a service server to handle the goal service request and a subscriber to listen to the goal result topic. The goals_reached and goals_cancelled variables are updated accordingly and printed out when the goal service is called. The node keeps running using rospy.spin() function.
