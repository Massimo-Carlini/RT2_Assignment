def user.odom_callback (   odom_msg)

This function receives an Odometry message, extracts the x, y position and linear velocity, creates a Position_velocity message and publishes it.

The function is a callback function that is called each time a new Odometry message is received by the subscriber. The function extracts the x, y position and x, y linear velocity from the message, creates a Position_velocity message, sets the extracted values in the message, and publishes the message to the "/position_velocity" topic.

Parameters
odom_msgThe Odometry message received by the subscriber.

def user.user ( )

This function displays a menu to the user and allows them to send a goal to the action server, cancel the current goal or exit the program.

The function prompts the user to enter a choice and performs the corresponding action based on the user's input. If the user selects to send a goal, the function gets the target coordinates from the user, creates a goal message with the target coordinates, and sends the goal to the action server. If the user selects to cancel the current goal, the function cancels the goal if there is an active goal, otherwise it displays a message that there is no active goal to cancel. If the user selects to exit the program, the function shuts down the rospy node.
