def print.info (   data)

This function processes the position and velocity information and prints the distance to the desired goal and average speed.

\detail This function processes the Position_velocity message received on the position_velocity topic. It takes in the data parameter which contains the current position and velocity of the robot. Inside the function, it first checks if the desired frequency has passed since the last print time by comparing the current time to the last print time. If it has, it gets the desired position d_x and d_y from the ROS parameter server using the rospy.get_param() function. Next, it calculates the distance between the current position and the desired position using the distance formula. It then calculates the velocity using the formula for the magnitude of a vector. Finally, it prints the distance and velocity to the console using the rospy.loginfo() function. It then updates the last print time to the current time.

Parameters
dataA Position_velocity message containing the current x, y position and linear velocity.
