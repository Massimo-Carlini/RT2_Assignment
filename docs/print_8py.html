Detailed Description

package assignment_2_2022

This Python node called "user" subscribes to the "/odom" topic to get the current position and velocity of a robot, creates a "Position_velocity" message containing the robot's position and velocity, and publishes it to the "/position_velocity" topic. Additionally, it provides a user interface for sending goals to the "/reaching_goal" action server and canceling active goals.
